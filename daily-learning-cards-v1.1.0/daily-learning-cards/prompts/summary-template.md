# 学习卡片摘要模板

## 输出格式

```
📝 昨日学习卡片 · {日期}

📊 **统计概览**
├─ 📚 核心主题：{主题数} 个
├─ 📋 重要决策：{决策数} 个
├─ 🐛 踩坑记录：{踩坑数} 个
└─ 🧠 新概念：{新概念数} 个

📚 **核心主题：**
{主题列表}

💭 **金句摘录：**
{金句列表}

🐛 **踩坑记录：**
{踩坑列表}

🧠 **新概念：**
{新概念列表}

📎 **完整卡片：** {卡片路径}

---
💃 金银 Planet · 自我提升部
```

## 列表格式

- 主题：📌 {主题名称}
- 金句：💭 {金句内容}
- 踩坑：🐛 {问题描述}
- 新概念：🧠 {术语}：{定义}

## 数量限制

- 主题：最多 5 个
- 金句：最多 3 条
- 踩坑：最多 3 条
- 新概念：最多 3 个
