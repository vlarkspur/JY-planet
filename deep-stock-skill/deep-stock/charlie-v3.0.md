### 2.3 验证标准

#### 专业报告可信度评估

**必须满足以下条件：**

| 评估维度 | 标准 | 说明 |
|----------|------|------|
| **来源可信** | 知名投行/咨询机构/行业协会 | J.P. Morgan, Goldman Sachs, IDC, Gartner, McKinsey |
| **发布时间** | 3个月内 | 超过3个月的数据可能过时 |
| **数据可验证** | 有明确数据来源 | 能追溯到原始数据出处 |
| **分析框架** | 有完整逻辑链条 | 不是简单数据罗列，有分析推理 |

**可信度分级：**
- ⭐⭐⭐⭐⭐ **高可信**：顶级投行/咨询机构，数据可验证
- ⭐⭐⭐⭐ **可信**：知名机构，数据基本可验证
- ⭐⭐⭐ **参考**：一般机构，数据需交叉验证
- ⭐⭐ **存疑**：来源不明，数据无法验证
- ⭐ **不可信**：明显错误，来源可疑

#### 信息交叉验证

**关键数据必须2个以上独立来源验证**

## 三、数据来源标注规范（V2.1核心，V3.0保留）

### 3.1 数据标注标准格式

**每个数据必须包含：**
```
数值（单位）[来源：机构名称，发布时间，获取时间]
```

**示例：**
- ✅ **正确：** "归母净利润92.86亿元[来源：比亚迪2025年报，2026-03-24发布，2026-05-09获取]"
- ❌ **错误：** "归母净利润约93亿"
- ❌ **错误：** "归母净利润92.86亿元（来自年报）"（无具体时间）

### 3.2 数据来源分级

| 等级 | 来源类型 | 可信度 | 使用规则 |
|------|---------|--------|---------|
| ⭐⭐⭐⭐⭐ | 官方原始数据（年报/公告/统计局） | 100% | 优先使用，可直接引用 |
| ⭐⭐⭐⭐ | 权威第三方（Wind/同花顺iFind/理杏仁） | 95% | 可使用，需标注平台和时间 |
| ⭐⭐⭐ | 专业媒体（财新/第一财经） | 80% | 需交叉验证，标注记者和日期 |
| ⭐⭐ | 自媒体/社群信息 | 50% | 仅作参考，不可作为决策依据 |
| ⭐ | 未标注来源 | 0% | **禁止使用** |

### 3.4 跨市场币种标注规则

**分析港股和A股时，每个估值数据必须标注币种，不能混用。**

| 市场 | 股票代码示例 | 币种 | 符号 |
|------|------------|------|------|
| A股（上交所） | 600XXX.SH / 601XXX.SH | 人民币 | ¥ 或 元 |
| A股（深交所） | 000XXX.SZ / 002XXX.SZ | 人民币 | ¥ 或 元 |
| 港股 | 0XXXX.HK | 港币 | HK$ 或 港元 |
| 美股 | XXXX.US | 美元 | $ 或 USD |
| ETF | XXXX.SZ/SH/HK | 跟随交易市场 | 按市场规则 |

**标注示例：**
- ✅ **A股：** "PE-TTM 22.5x，目标价 ¥48.2/股"
- ✅ **港股：** "PE-TTM 22.5x，目标价 HK$48.2/股"
- ✅ **美股：** "PE-TTM 22.5x，目标价 $48.2/股"
- ❌ **错误：** "目标价 48.2"（无币种，无法判断是人民币还是港币）

### 3.3 禁止行为清单

- ❌ **禁止编造数据** — 不确定时必须说"I don't know"
- ❌ **禁止虚假标注** — 不能标注"国家统计局"却不从stats.gov.cn核实
- ❌ **禁止模糊表述** — "约""大概""可能"必须有误差范围
- ❌ **禁止过时数据** — 超过3个月的数据必须标注"[过时]"

### 3.4 数据核查清单（每次分析前必须勾选）

- [ ] 所有数值都有明确来源标注？
- [ ] 所有"国家统计局"数据都从stats.gov.cn核实？
- [ ] 所有财报数据都从年报/季报原文提取？
- [ ] 所有第三方数据都标注了平台和时间？
- [ ] 关键数据（影响决策）有2个以上独立来源？
- [ ] 方向性数据（正负/增减）逐字核对原始来源？

**未完成核查 → 分析结论置信度降为"存疑"**

## 四、评分量表（V2.1核心，V3.0保留）

### 4.1 评分原则

**每个评分必须有量化依据，禁止主观拍脑袋**

### 4.2 国际局势评分（0-10分，权重15%）

| 评分 | 定义 | 量化标准 | 典型情景 |
|------|------|---------|---------|
| 9-10 | 极安全 | 地缘平稳+美元宽松+资源充足 | 2024年9月美联储降息周期 |
| 7-8 | 较安全 | 局部摩擦但可控+美元中性 | 当前（2026年5月） |
| 5-6 | 中性 | 地缘紧张+美元波动+资源涨价 | 2022年俄乌冲突初期 |
| 3-4 | 较危险 | 地缘冲突升级+美元紧缩 | 2022年3月加息周期 |
| 0-2 | 极危险 | 全面冲突+美元危机+资源断供 | 假设台海/中东全面战争 |

**评分公式：**
```
地缘风险分 = 冲突烈度(0-3) + 关税影响(0-2) + 物流风险(0-2) + 制裁风险(0-3)
国际局势得分 = 10 - 地缘风险分
```

### 4.3 国内局势评分（0-10分，权重25%）

| 评分 | 定义 | 量化标准 | 典型情景 |
|------|------|---------|---------|
| 9-10 | 强劲复苏 | GDP>5.5%+CPI>2%+PMI>52% | 2021年疫后复苏 |
| 7-8 | 温和复苏 | GDP 5%+CPI 1-2%+PMI 50-52% | 当前（2026年Q1 GDP+5.0%） |
| 5-6 | 弱复苏 | GDP 4-5%+CPI<1%+PMI 48-50% | 2025年通缩压力期 |
| 3-4 | 滞胀风险 | GDP<4%+CPI>3%+PMI<48% | 假设滞胀情景 |
| 0-2 | 衰退 | GDP<3%+CPI<0%+PMI<45% | 假设硬着陆 |

**评分公式：**
```
经济动能分 = GDP增速(0-3) + PMI景气度(0-3) + 消费增速(0-2) + 政策力度(0-2)
国内局势得分 = 经济动能分
```

### 4.4 行业景气评分（0-10分，权重25%）

| 评分 | 定义 | 量化标准 | 典型行业 |
|------|------|---------|---------|
| 9-10 | 爆发期 | 渗透率<30%+增速>30%+无价格战 | 2021年新能源车 |
| 7-8 | 成长期 | 渗透率30-50%+增速15-30% | 当前新能源车 |

**评分公式：**
```
行业景气分 = 增速排名(0-3) + 渗透率空间(0-3) + 竞争格局(0-2) + 技术迭代风险(0-2)
行业景气得分 = 行业景气分
```

### 4.5 公司基本面评分（0-10分，权重25%）

| 评分 | 定义 | 量化标准 | 典型公司 |
|------|------|---------|---------|
| 9-10 | 卓越 | ROE>20%+毛利率>30%+现金流>净利润 | 茅台、苹果 |
| 7-8 | 优秀 | ROE 15-20%+毛利率 20-30%+现金流匹配 | 比亚迪（2024年前） |

**评分公式：**
```
基本面分 = ROE得分(0-3) + 毛利率得分(0-3) + 现金流得分(0-2) + 成长性得分(0-2)
公司基本面得分 = 基本面分
```

### 4.6 估值评分（0-10分，权重10%）

| 评分 | 定义 | 量化标准 | 典型估值 |
|------|------|---------|---------|
| 9-10 | 极度低估 | PE<历史10%分位+PEG<0.5 | 2024年9月港股 |

```

### 4.7 综合评分计算

```
总分 = 国际局势×15% + 国内局势×25% + 行业景气×25% + 公司基本面×25% + 估值×10%

## 七、估值分析模块（V3.0新增）

### 4.1 核心原则

> **单一估值方法容易陷入"估值陷阱"，必须用多种方法交叉验证。**

**四种估值方法：**
- **PE-TTM**：市盈率，盈利稳定公司
- **PEG**：市盈增长比率，成长股
- **DCF**：现金流折现，所有公司（理论上）
- **SOTP**：分部加总，多元化业务公司

### 4.2 估值方法适用场景

| 估值方法 | 适用公司 | 不适用 | 优先级 |
|---------|---------|--------|--------|
| PE-TTM | 盈利稳定、无重大周期波动 | 亏损/盈利波动极大 | ⭐⭐⭐⭐⭐ |
| PEG | 成长股（增速>15%） | 成熟/衰退行业 | ⭐⭐⭐⭐⭐ |
| DCF | 所有公司（理论上） | 现金流预测难度大 | ⭐⭐⭐⭐ |
| SOTP | 多元化业务公司 | 单一业务公司 | ⭐⭐⭐⭐ |

### 4.3 PE-TTM估值

**公式：** PE = 股价 / EPS-TTM

**步骤：**
1. 计算当前PE-TTM = 股价 / 过去12个月每股收益
2. 对比历史分位（近5年）
3. 对比行业平均PE
4. 确定合理PE区间

**数据来源要求：**
- 股价：[来源：XX交易所,日期]
- EPS：[来源：XX财报,日期]
- 历史PE：[来源：Wind/同花顺,时间范围]

### 4.4 PEG估值

**公式：** PEG = PE / 盈利增速(G)

**标准：**
- PEG<0.5：极度低估
- PEG 0.5-0.8：显著低估
- PEG 0.8-1.2：合理
- PEG>1.2：偏高

**数据来源要求：**
- 预期增速必须有来源（公司指引/分析师预期/行业报告）
- 无来源 → 标注"[估算，待验证]"，置信度降级

### 4.5 DCF估值（简化版）

**公式：** 企业价值 = Σ[FCF_t / (1+WACC)^t] + 终值/(1+r)^n

**关键参数：**
- WACC：加权平均资本成本（必须展示计算过程）
- 永续增长率g：通常2-3%
- 预测期：3-5年

**敏感性分析：**
- 乐观：增速+2%，WACC-1%
- 基准：中性假设
- 悲观：增速-2%，WACC+1%

### 4.6 SOTP估值

**步骤：**
1. 业务分拆（汽车/电池/电子）
2. 各业务单独估值（净利润×PE）
3. 加总并调整（净债务/现金）
4. 考虑折价/溢价

**数据来源要求：**
- 分部利润：公司披露或估算依据
- 可比公司PE：明确来源
- 估算数据 → 标注来源，降级置信度

### 4.7 估值交叉验证

| 方法 | 估值 | 数据来源 | 置信度 |
|------|------|---------|--------|
| PE-TTM | XX元 | 股价/EPS来源 | ⭐⭐⭐⭐ |
| PEG | XX元 | 增速来源 | ⭐⭐⭐ |
| DCF | XX元 | WACC计算/假设 | ⭐⭐⭐ |
| SOTP | XX元 | 分部利润/可比公司 | ⭐⭐⭐⭐ |
| **综合** | **XX-XX元** | — | — |

**决策：**
- 差异<20%：取区间中值
- 差异>20%：分析原因，保守取值

## 八、因果回路图（V3.0新增）

### 5.1 核心原则

> **用文字描述传导链条不够直观，必须用系统动力学可视化。**

**符号说明：**
- ➕：同向变化（A↑→B↑）
- ➖：反向变化（A↑→B↓）
- ○：延迟
- ⇄：反馈回路

### 5.2 增强回路（正反馈）

**规模效应回路：**
```
销量↑ ➕ 规模效应↑ ➕ 成本↓ ➕ 价格竞争力↑ ➕ 销量↑
```

**出口增长回路：**
```
出口占比↑ ➕ 海外收入↑ ➕ 品牌力↑ ➕ 出口竞争力↑ ➕ 出口占比↑
```

### 5.3 平衡回路（负反馈）

**价格战回路：**
```
销量↑ ➕ 价格战↑ ➖ 毛利率↓ ➖ 利润↓ ➖ 研发↓ ➖ 技术领先↓ ➖ 销量↓
```

**技术颠覆回路：**
```
固态电池落后↑ ➕ 竞争力↓ ➖ 销量↓ ➖ 利润↓ ➖ 研发↓ ➕ 技术差距↑
```

### 5.4 应用

**识别杠杆点：**
- 高杠杆：技术领先→规模效应，出口占比→出口增长
- 风险点：价格战→研发减少→技术落后

**识别干预点：**
- 可干预：央行降准→融资成本↓→价格战缓冲
- 不可干预：美联储政策、技术路线选择

## 九、贝叶斯概率情景推演（V3.0核心升级）

### 6.1 核心原则

> **概率必须随新证据动态更新，拒绝主观拍脑袋。**

**贝叶斯公式：**
```
后验概率 = (先验概率 × 似然度) / 证据概率

简化：新概率 = 旧概率 × 证据支持度
```

### 6.2 6种核心情景（先验概率）

| 情景 | 初始概率 | 核心假设 | 触发条件 |
|------|---------|---------|---------|
| 情景1：全面乐观 | 10% | 联储降息+国内刺激+技术突破 | 小概率 |
| 情景2：全面悲观 | 10% | 美元紧缩+国内衰退+价格战 | 小概率 |
| 情景3：国际紧+国内松 | 20% | 联储降息但国内对冲 | 当前最可能 |
| 情景4：国内紧+公司好 | 15% | 政策收紧但业绩拐点 | 结构性 |
| 情景5：基准情景 | 30% | 弱复苏持续+行业平稳 | **基准** |
| 情景6：出口强+国内弱 | 15% | 海外需求爆发+国内疲软 | 比亚迪特有 |

### 6.3 证据分级与似然度

| 证据等级 | 定义 | 权重 | 示例 |
|---------|------|------|------|
| 🔴 强证据 | 直接改变基本面 | ±15-25% | 业绩暴雷/技术突破 |
| 🟡 中证据 | 显著影响假设 | ±5-15% | 月度数据超预期 |
| 🟢 弱证据 | 边际影响 | ±2-5% | 市场情绪变化 |
| ⚪ 噪音 | 无实质影响 | 0% | 日常波动 |

### 6.4 概率更新示例

**初始：** 情景6 = 15%

**新证据：** Q1出口+15%（超预期，中证据）

**似然度：** P(+15%出口|情景6) = 高（+15%支持）

**计算：**
```
情景6新概率 = 15% × 1.15 = 17.25%
归一化后 ≈ 20%
```

**更新后：** 情景6 = 20%（+5%）

### 6.5 期望收益计算

```
期望收益 = Σ(情景概率 × 情景收益)

示例：
= 10%×(+50%) + 10%×(-40%) + 20%×(+10%) 
  + 15%×(+5%) + 30%×(+8%) + 15%×(+20%)
= +8.5%
```

### 6.6 概率更新触发条件

**自动触发（必须更新）：**
- 季度财报发布
- 月度宏观数据（CPI/PMI/GDP）
- 重大政策announcement
- 技术突破/落后确认

**手动触发（建议更新）：**
- 周度股东户数变化
- 月度销量数据
- 机构持仓变化

### 6.7 输出模板

```markdown
### 情景概率更新（贝叶斯更新）

**触发证据：** [证据描述]

| 情景 | 原概率 | 证据支持度 | 新概率 | 变化 | 投资建议 |
|------|--------|-----------|--------|------|---------|
| 情景6 | 15% | +15% | 20% | +5% | 加仓3% |

**期望收益：** +8.5%
**决策：** 加仓至5%
```

| **管理层** | 如果创始人退休，公司还能运转吗？ | 小米：雷军个人色彩浓，风险中等 |
| **财务** | 如果营收-30%，公司会破产吗？ | 比亚迪：负债率71%，但主要是应付账款 |
| **估值** | 如果PE再跌50%，逻辑还成立吗？ | 比亚迪：PE 12仍合理（增速10%+） |

### 7.3 认知偏差检查（芒格25种误判）

**必查5种：**
1. **喜欢/热爱倾向** — 是否因为喜欢品牌而忽视风险？
2. **避免不一致性倾向** — 是否因为已持仓而倾向加仓？
3. **社会认同倾向** — 是否因为"别人都买"而跟风？
4. **稀缺性错误倾向** — 是否被"错过就没机会"忽悠？
5. **权威错误倾向** — 是否因为"巴菲特说"而盲从？
