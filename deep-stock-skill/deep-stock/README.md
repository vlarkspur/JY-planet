# Deep Stock 🎯
## 一套框架，穿透噪音

专业股票分析框架——从宏观到估值的6步深度分析，基于 Charlie V3.0 方法论（致敬查理·芒格的多元思维模型）。

## 安装

```bash
clawhub install deep-stock
```

## 使用方法

安装后，直接告诉你的 AI：

- "分析比亚迪"
- "用 Deep Stock 分析 002594.SZ"
- "分析小米"

你会得到一份完整的结构化分析报告，涵盖宏观、行业、估值、概率和决策。

## 包含文件

- `SKILL.md` — 主技能定义和触发说明
- `charlie-v3.0.md` — Charlie V3.0 完整框架（12章）
